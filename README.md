# TagRacer - Multiplayer NFC Game for Flipper Zero

TagRacer ist ein spannendes Multiplayer-Spiel für den Flipper Zero, das die NFC-Fähigkeiten des Geräts nutzt, um eine interaktive und wettbewerbsorientierte Spielerfahrung zu schaffen.

## Projektstruktur

- `/flipper-app`: Flipper Zero Anwendung für NFC-Scanning
- `/bridge`: Bridge-Software für WiFi Dev Board/ESP32
- `/server`: Backend Server
- `/web`: Web Interface für Scoreboard

## Features

- Echtzeit NFC-Tag Scanning
- Live Scoreboard
- Weltweiter Multiplayer-Modus
- Zeitbasierte Spielrunden
- Anpassbares Punktesystem

## Setup

### Voraussetzungen

- Flipper Zero mit aktueller Firmware
- WiFi Dev Board, Raspberry Pi oder ESP32
- Internet-Verbindung
